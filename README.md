## 📌 Sobre o Projeto

**AdaptFin** é um assistente financeiro inteligente que se adapta automaticamente ao seu comportamento de gastos. Diferente dos apps tradicionais que exigem configuração manual de metas e categorias, o AdaptFin observa seus padrões e oferece insights personalizados.

### 🎯 O Problema que Resolve

> *"A maioria dos apps de finanças foi feita para quem já tem disciplina financeira. Quem mais precisa de ajuda é exatamente quem abandona esses apps na segunda semana."*

### 💡 Nossa Solução

| Problema | Solução AdaptFin |
|----------|------------------|
| Configuração complexa | ✅ Sem configuração manual - sistema aprende com você |
| Metas difíceis de seguir | ✅ Recomendações adaptativas baseadas no seu histórico |
| Falta de insights | ✅ IA detecta padrões e sugere melhorias |
| Dados presos no app | ✅ Exportação CSV/Excel, backup local |
| Abandono após 2 semanas | ✅ Interface intuitiva com tutorial e dicas |

## ✨ Funcionalidades

### Core Features
- 📊 **Dashboard Interativo** - Métricas em tempo real, gráficos dinâmicos
- 💰 **Múltiplas Entradas** - CSV, Receitas manuais, Despesas Fixas, Gastos Variáveis
- 🔄 **Contas Recorrentes** - Configure uma vez, geramos automaticamente
- 📈 **Relatórios Completos** - Mensal, Anual, Por Categoria, Projeções
- 🤖 **Machine Learning** - Previsões, anomalias, insights personalizados
- 💾 **Persistência** - Salvamento automático, backups, exportação CSV/Excel
- 🎨 **Dark Mode** - Interface clara ou escura, à sua escolha
- 🎓 **Tutorial Interativo** - Tour guiado na primeira execução
- 💡 **Tooltips Contextuais** - Dicas em cada campo

### Funcionalidades por Módulo

| Módulo | Funcionalidades |
|--------|-----------------|
| **Dashboard** | Visão geral, métricas KPIs, gráficos de evolução, top gastos |
| **Transações** | Listagem completa, filtros, categorização automática |
| **Relatórios** | Análise mensal, consolidado anual, distribuição por categoria |
| **ML Insights** | Detecção de anomalias, previsões, perfil do usuário |
| **Configurações** | Limites financeiros, alertas, tema escuro/claro |
